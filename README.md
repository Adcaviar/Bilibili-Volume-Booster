# Bilibili Volume Booster

一个用于哔哩哔哩网页版的 Chromium / Edge 浏览器扩展（Manifest V3），可将当前标签页音频放大到播放器 100% 以上。

## 功能特性

- **哔哩哔哩专属**：仅对 `*.bilibili.com` 页面生效。
- **标签页级增强**：按标签页独立开启/关闭，关闭标签页或跳转到非哔哩哔哩页面时自动释放音频流。
- **倍率调节**：支持 **1% – 500%** 增强倍率（默认 200%）。
- **智能音量**（默认关闭）：开启后根据源视频响度实时自动调节倍率，无需手动拖动滑块。
- **手动模式**（默认）：关闭智能音量时，使用固定倍率增强，可在弹窗中拖动滑块或点击数值输入。
- **弹窗快捷开关**：工具栏图标一键开启/关闭增强。
- **快捷键**：默认 `Alt + Shift + U` 切换当前哔哩哔哩标签页增强状态。
- **设置页**：默认倍率、智能音量、快捷键、通知开关等，设置通过 `chrome.storage.sync` 同步。
- **系统通知**：开启/关闭增强时可选择是否弹出桌面通知。

## 安装

1. 打开浏览器扩展管理页：
   - Chrome：`chrome://extensions`
   - Edge：`edge://extensions`
2. 开启 **开发者模式**。
3. 点击 **加载已解压的扩展程序**。
4. 选择本项目根目录。

## 使用说明

### 弹窗

1. 打开哔哩哔哩并播放视频/音频。
2. 点击工具栏中的扩展图标打开弹窗。
3. 调整 **增强倍率**（智能音量关闭时可用）。
4. 切换 **智能音量** 开关（与设置页同步）。
5. 点击 **开启增强** / **关闭增强**。

也可点击右上角齿轮进入设置页。

### 智能音量

默认关闭。在弹窗或设置页开启后，扩展会实时采样当前标签页音频响度，并自动计算输出倍率：

| 源视频响度 | 行为 |
|-----------|------|
| 偏小 | 自动提高倍率（最高约 250%） |
| 正常 | 约 100% |
| 偏大 | 自动降低倍率（最低约 25%） |
| 静音/无声 | 固定 100%，避免放大底噪 |

此时滑块会锁定，倍率数字随响度实时变化。关闭智能音量后，倍率与音量会恢复到开启智能音量之前的值。

> 智能音量完全依据源视频响度计算，**不受**设置页「默认增强倍率」影响。

### 手动模式

关闭智能音量后：

- 使用弹窗滑块或点击倍率数字手动输入（1% – 500%）。
- 默认倍率可在设置页配置，弹窗与设置页共享该值。
- 增强开启后拖动滑块，松手时立即应用到当前标签页。

### 快捷键

默认快捷键：`Alt + Shift + U`

在哔哩哔哩标签页按下即可开关增强。扩展通过两种方式响应快捷键：

1. **浏览器命令**（`chrome://extensions/shortcuts` 中绑定的扩展命令）
2. **页面内监听**（设置页录制的快捷键保存在 storage 中，由 content script 在哔哩哔哩页面拦截按键）

若两种方式绑定的快捷键一致，则只走浏览器命令，避免重复触发。

**修改快捷键**

- 在设置页点击 **点击设置快捷键** 录制新组合键（Chrome / Edge 支持）。
- 点击 **清除** 可恢复为默认 `Alt + Shift + U`。
- 修改后请 **刷新哔哩哔哩页面** 使页面内监听生效。
- 若使用浏览器命令，建议同时在 `chrome://extensions/shortcuts`（Edge 为 `edge://extensions/shortcuts`）确认扩展命令 **开关哔哩哔哩音量增强** 已正确绑定。

> 首次安装时，manifest 中的 `suggested_key` 会尝试自动绑定 `Alt + Shift + U`。若 shortcuts 页显示为空（与其他扩展或浏览器快捷键冲突），可在该页面手动设置，或在设置页录制快捷键后刷新哔哩哔哩页面使用。

### 设置页

| 选项 | 说明 |
|------|------|
| 默认增强倍率 | 关闭智能音量时使用的固定倍率（默认 200%） |
| 智能音量 | 是否启用响度自适应（默认关闭） |
| 快捷键开关 | 查看、录制或清除快捷键 |
| 开关时显示系统通知 | 增强开/关时是否弹出通知 |

## 工作原理

```
哔哩哔哩标签页
    ↓ tabCapture
Offscreen 文档（Web Audio）
    ↓ AnalyserNode 采样响度
    ↓ GainNode 放大
系统音频输出
```

1. `background.js` 通过 `chrome.tabCapture` 获取当前标签页音频流。
2. `offscreen.js` 在 Offscreen 文档中建立 Web Audio 处理链：`source → analyser → gain → destination`。
3. **智能音量开启**时，每帧根据 RMS 响度计算倍率并实时应用。
4. **智能音量关闭**时，使用用户设定的固定倍率。

## 注意事项

- 倍率过高可能导致破音或失真，建议从 **150% – 250%** 开始尝试。
- 增强的是当前标签页的音频输出，不会修改哔哩哔哩播放器内的音量 UI。
- 多个标签页可同时开启增强，各自独立计算倍率。
- 面向 Chromium Manifest V3 实现，Firefox 需单独适配。
- 修改代码后需在 `chrome://extensions` 中 **重新加载扩展** 才能生效。

## 权限说明

| 权限 | 用途 |
|------|------|
| `tabCapture` | 捕获哔哩哔哩标签页音频 |
| `offscreen` | 在 Offscreen 文档中处理音频 |
| `storage` | 保存用户设置（倍率、智能音量等） |
| `scripting` | 安装/更新时向已打开的哔哩哔哩页面注入脚本 |
| `tabs` / `activeTab` | 获取当前标签页信息 |
| `notifications` | 增强开/关时的桌面通知 |
| `host_permissions: *.bilibili.com` | 限制仅在哔哩哔哩站点工作 |

## 开发

本项目为纯前端扩展，无需构建步骤。修改 `src/` 下文件后，在扩展管理页点击 **重新加载** 即可调试。

主要调参位置（智能音量）在 `src/offscreen.js` 顶部常量：

| 常量 | 含义 |
|------|------|
| `NOMINAL_RMS` | 标准响度参考值 |
| `ADAPT_MIN` / `ADAPT_MAX` | 自动倍率下限 / 上限 |
| `SILENCE_RMS` | 低于此值视为无声 |
| `ADAPT_TICK_MS` | 响度采样间隔（约 60fps） |
